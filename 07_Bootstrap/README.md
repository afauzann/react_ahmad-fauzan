## Bootstrap

**Bootstrap** yaitu  framework HTML, CSS, dan JavaScript yang berfungsi untuk mendesain website responsive dengan cepat dan mudah.

## Kenapa Bootstrap ?

* Alasan kenapa Bootstrap

    * Ramah untuk Pemula
    * Grid System yang Canggih
    * Open Source
    * Mendukung Responsif
    * Dokumentasi Lengkap
    * Up to Date
